# 系统架构

```text
[ Web / 管理端 ] ──┐
[ 移动端 App   ] ──┼──▶ [ API 网关 ] ──▶ [ 业务服务 ] ──▶ [ 数据库 ]
```

> 需要可交互架构图时，可导出 PNG/SVG 到 `docs/assets/diagrams/` 引用，或自行接入 Mermaid 插件。

## 关键链路

1. **同步请求路径**：客户端 → 网关 → 服务 → 数据存储。
2. **异步事件**：说明消息队列主题、重试与死信策略（链接到运维 Runbook）。
