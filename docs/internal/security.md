# 安全与合规

## 数据分级（示例）

| 级别 | 描述 | 存储与传输 |
| --- | --- | --- |
| L3 机密 | 密钥、生产凭据 | 仅密钥管理系统；禁止进 Wiki |
| L2 内部 | 架构、接口、运维手册 | HTTPS + 内网或 SSO |
| L1 公开 | 对外的产品介绍 | 独立站点 |

## 访问控制

- 文档站本身建议由 **Nginx + SSO（OIDC/LDAP）** 或 **VPN 内网** 保护，不要仅依赖「链接保密」。
- 对外 API 的认证方式见 [认证与鉴权](../api/authentication.md)。

## 事件上报

发现敏感信息误发、权限异常，按公司安全流程上报，并在此文追加「事后复盘」小节（不含敏感细节）。
